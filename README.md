# 🏦 Bank Card Project

Система управление банковскими картами на Python.
Учебный пет-проект для отработки ооп, архитектуры и также для изучения новых технологий.


## 📁 Структура проекта

bank-card-project/
├── models/          #Модели данных 
├── services/        #Бизнес-логика
├── factory/         #Фабрика карт
├── dop_work/        #Вспомогательное 
└── main.py          #Точка входа




## ⚙️ Возможности

- Пополнение и снятие с PIN-кодом
- Переводы между картами с атомарным откатом
- Аналитика транзакций (топ трат, средний чек, отчёт по месяцам)
- Сравнение карт
- Логирование всех операций




## 🛠️ Технологии

- Python 3.12
- ООП, паттерны: Singleton, Factory, Mixin, Decorator
- Context Manager для атомарных транзакций
- logging, uuid, datetime, collections
